# Danh Mục Thuật Ngữ và Từ Viết Tắt

## I. Danh Mục Thuật Ngữ

| Thuật ngữ | Ý nghĩa |
| :--- | :--- |
| **Component** | Thành phần chức năng trong Unity, mỗi component định nghĩa một hành vi hoặc thuộc tính riêng biệt có thể gắn vào GameObject. |
| **Component-Based** | Phương pháp thiết kế phần mềm hoặc game dựa trên các thành phần độc lập (Component). |
| **Controller** | Bộ điều khiển vật lý cho thiết bị VR. |
| **GameObject** | Đối tượng cơ bản trong Unity, đóng vai trò là khối xây dựng cho mọi thành phần trong màn chơi. |
| **Gesture** | Cử chỉ tay được nhận diện để tương tác trong game VR. |
| **Hand Tracking** | Công nghệ theo dõi chuyển động bàn tay và ngón tay. |
| **HUD** | Giao diện hiển thị thông tin trên màn hình (Heads-Up Display). |
| **Meta Quest** | Dòng kính thực tế ảo độc lập của Meta (Oculus Quest cũ). |
| **MonoBehaviour** | Lớp cơ bản cho các script trong Unity Engine. |
| **NavMesh Agent** | Công nghệ điều hướng trí tuệ nhân tạo trong môi trường 3D. |
| **OpenXR** | Chuẩn giao diện lập trình ứng dụng mở cho thực tế ảo và thực tế tăng cường. |
| **ĐATN** | Đồ án tốt nghiệp. |

<br>

## II. Danh Mục Từ Viết Tắt

| Từ viết tắt | Ý nghĩa |
| :--- | :--- |
| **FPS** | Bắn súng góc nhìn thứ nhất (First-Person Shooter). |
| **IDE** | Môi trường phát triển tích hợp (Integrated Development Environment). |
| **SDK** | Bộ công cụ phát triển phần mềm (Software Development Kit). |
| **UI** | Giao diện người dùng (User Interface). |
| **UML** | Ngôn ngữ mô hình hóa thống nhất (Unified Modeling Language). |
| **UX** | Trải nghiệm người dùng (User Experience). |
| **VR** | Thực tế ảo (Virtual Reality). |
| **XR** | Thực tế mở rộng (Extended Reality). |
| **XRI** | Bộ công cụ tương tác XR (XR Interaction Toolkit). |
