# DANH MỤC THUẬT NGỮ

| Thuật ngữ | Nghĩa đầy đủ tiếng Anh | Ý nghĩa / Giải thích tiếng Việt |
|---|---|---|
| **ASL** | American Sign Language | Ngôn ngữ ký hiệu Mỹ |
| **VR** | Virtual Reality | Thực tế ảo |
| **XR** | Extended Reality | Thực tế mở rộng (bao gồm VR, AR, MR) |
| **HMD** | Head-Mounted Display | Thiết bị hiển thị đeo đầu (Kính thực tế ảo) |
| **SDK** | Software Development Kit | Bộ công cụ phát triển phần mềm |
| **SO** | ScriptableObject | Đối tượng lưu trữ dữ liệu dạng tài sản độc lập trong Unity |
| **FSM** | Finite State Machine | Máy trạng thái hữu hạn |
| **EdTech** | Educational Technology | Công nghệ giáo dục |
| **UI** | User Interface | Giao diện người dùng |
| **UX** | User Experience | Trải nghiệm người dùng |
| **FPS** | Frames Per Second | Số khung hình hiển thị trên giây (Đo hiệu năng) |
| **Controller-free** | Controller-free Interaction | Tương tác không cần tay cầm (dùng tay vật lý) |
| **Event-Driven** | Event-Driven Architecture | Kiến trúc hướng sự kiện |
| **$1 Recognizer** | Dollar One Unistroke Recognizer | Thuật toán nhận dạng nét vẽ đơn góc tự do |
| **SUS** | System Usability Scale | Thang đo độ khả dụng của hệ thống |
